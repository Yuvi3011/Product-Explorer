# Product-Explorer


##  Description
Product Explorer is a web application that allows users to browse products using a public API. Users can search, filter, and sort products based on different criteria.

##  API Used
Fake Store API - https://fakestoreapi.com/products

##  Features
- Display products dynamically
- Search products by name
- Filter products by category
- Sort products by price
- Responsive design

##  Technologies Used
- HTML
- CSS
- JavaScript (Fetch API, Array Methods)

##  How to Run
1. Clone the repository
2. Open index.html in browser
